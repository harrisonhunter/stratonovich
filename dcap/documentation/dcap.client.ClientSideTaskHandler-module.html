Module ClientSideTaskHandler

source code

This module contains methods for dealing with tasks received from the server.

It takes the approach that each task will consist out of a script and possibly a zipped data folder. Both the script and the zipped folder will be stored in temp asa data. The zipped folder will be extracted and the script executed with full privilege. If you wish to take a different approach of handing tasks to the server, you can replace this module

Author: Alexander Waldin
